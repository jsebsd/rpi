The files in this directory are attributed to Adrian Rosebrock, PyImageSearch.

This is an effective motion detection system that I was trying to use to help implement my own
fall and motion detection system. 

His project can be accessed here :
http://www.pyimagesearch.com/2015/06/01/home-surveillance-and-motion-detection-with-the-raspberry-pi-python-and-opencv/

The command to make the system run is
> sudo python pi_surveillance.py --conf conf.json
